# 💸 Expense Tracker Frontend

A modern expense tracking web application built with React and Vite.  
Track expenses, visualize spending, and manage your finances with an intuitive dashboard.

## 🚀 Live Demo

🌐 https://expense-tracker-frontend-sage-chi.vercel.app/

## ✨ Features

- User authentication
- Add, edit, and delete expenses
- Expense analytics dashboard
- AI-powered expense overview
- Responsive modern UI
- Secure API integration

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- Axios
- Django REST Framework Backend

## ⚙️ Installation

```bash
git clone https://github.com/nishana-codes/expense-tracker-frontend.git

cd expense-tracker-frontend

npm install

npm run dev
🌐 Environment Variables

Create a .env file in the root directory:

VITE_API_URL=http://localhost:8000
📦 Build for Production
npm run build
